THE RULES OF PICROSS
  1. Squares are either filled or empty.
  2. The numbers before each row or on top of that column indicate exactly the pattern of filled squares in that row or column.
  3. For instance, if the number says “4”, that means there are four filled squares in a row in that column, and the rest are empties.
  4. If the number says “4 5”, that means there are four filled squares in a row in that column, some number of empty squares, and then five filled squares in a row (and then empties, in any squares are left).
  5. “2 1 2 1” means the pattern in that row is two filled squares, some number of empties, 1 filled square, some number of empties, 2 filled squares in a row, some number of empties, and then one filled square (and then empties, if any squares are left).
  6. It’s a logic puzzle. Your goal is to figure out the position of filleds and empties in the whole grid. At the end of each level, you’re rewarded with the semi-abstract picture you just uncovered. If you need more detailed explanation, just push Wikipedia button. Enjoy!
